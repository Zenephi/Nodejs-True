How to run the Node js

after you download this file or copy this do this:

1. Open the Visual Studio Code

2. Open folder then find the CHAT SYSTEM

3. Then open the terminal in the Visual Studio Code

4. The type this in the terminal or Copy
      npm init -y
      npm install express
      npm install socket.io
      npm install ejs

5. then after install the pkg then type this in the terminal
      npm server

6. then go to the browser type this or copy
      http://localhost:3000
